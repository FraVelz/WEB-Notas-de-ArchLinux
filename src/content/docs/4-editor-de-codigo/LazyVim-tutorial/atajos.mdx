## Neo-tree (explorador de archivos)

| Tecla | Acción                     |
| ----- | -------------------------- |
| `r`   | Renombrar archivo/carpeta  |
| `c`   | Copiar                     |
| `x`   | Cortar                     |
| `p`   | Pegar en la carpeta actual |
| `y`   | Copiar la ruta del archivo |
| `q`   | Cerrar la barra lateral    |
| `d`   | Eliminar archivo o carpeta |

### Crear archivo o carpeta

Presiona `a`. Neo-tree te pedirá un nombre:

* Si escribes `archivo.cpp` creará un archivo.
* Si escribes `nueva_carpeta/` (con / al final) creará una carpeta.

---

## Neovim

| Atajo    | Acción                    |
| -------- | ------------------------- |
| `Ctrl+p` | Buscar archivo (edición rápida) |
| `Ctrl+n` | Abrir barra de Neo-tree   |
